# Amazon EC2 Basics

Amazon EC2 = Virtual servers in the cloud.

### Key Concepts:
- **AMI** → Template for OS and configuration
- **Instance** → The virtual machine
- **Instance Type** → Defines CPU, RAM, network performance
- **Security Groups** → Virtual firewalls for instances
- **Key Pairs** → SSH/RDP access credentials

### Use Cases:
- Web servers
- Application servers
- Batch processing
- Development/testing
